# Channel Selection Matrix

Do not treat all platforms as one channel. Each has different user intent, content
format, and conversion path. **Rule: pick 1–2 channels first, go deep, then expand.**

**Surface gate:** channels must match the product's `project.surfaces`. **App Store
Search / ASO is N/A for products with no app-store surface** (`ios`/`android`/
`mac_app_store`) — do not recommend it for a website-only product; its conversion target
is the website, not an App Store page. Only offer app-store channels when the product
actually ships through an app store.

| Channel | User Intent | Best Content Type | Main Risk |
|---|---|---|---|
| Google Search | High | SEO pages, comparison pages, tutorials | Slow, competitive — but high intent |
| App Store Search | Very high | ASO, screenshots, app name, subtitle | Limited space, ranking uncertainty |
| YouTube | Medium–high | Demos, tutorials, reviews | Heavier production effort |
| Instagram / TikTok | Low–medium | Awareness, emotional hooks, short demos | Attention, often low intent |
| Reddit | High trust, high skepticism | Helpful answers, problem discussions | Hostile to obvious promotion |
| Product Hunt | Early adopters | Launch, credibility, feedback | Weak without existing community |
| Twitter/X | Founder/product audience | Build-in-public, updates, storytelling | Requires consistent presence |
| LinkedIn | Professional / B2B | B2B use cases, workflow content | Not ideal for consumer apps |
| Owned Niche Directory | High comparison intent | Editorial "best tools" lists, curated comparisons | Build/maintenance cost; compounds long-term |
| Third-Party Directories | Trust + referral | Strategic listings where users already search | Spam farms waste time |

## How the audit scores channel fit

For each candidate channel:
1. **Intent match** — does the channel's native intent match the user's dominant
   intent class (from dim 2)? High-intent products misplaced on low-intent channels
   (e.g. a high-purchase-intent tool relying only on TikTok awareness) lose points.
2. **Format fit** — can the team realistically produce the channel's best content type?
3. **Conversion path** — is there a clear route from channel → landing/App Store → install?
4. **Focus** — more than 2 primary channels = shallow effort penalty.

Good channel selection = **1–2 channels**, each with intent match + producible format
+ a named conversion path, justified by where users already show intent (not by hype).

## Channel rule violations (red flags)

- "We'll post everywhere" → shallow-effort penalty.
- Treating all social platforms as one channel with the same content/expectations.
- Choosing a channel because it's trendy, not because users show intent there.
- Product Hunt / launch platform treated as the main channel with no existing audience.
